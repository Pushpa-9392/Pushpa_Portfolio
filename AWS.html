Home made Pickle and Snacks (AWS)

Cloud-based Culinary Platform

Mission of the Project

To showcase authentic, handcrafted pickles and snacks through a cloud-based platform built using HTML, CSS, and JavaScript.

Introduction

This project is a cloud-based platform showcasing handcrafted pickles and snacks. It is built using HTML, CSS, and JavaScript to provide an interactive and visually appealing user experience.


Problem Statement


However, Flipkart does not provide an easily accessible public API for retrieving such data. This creates a challenge in efficiently gathering large datasets in a structured format.


Therefore, there is a need for an automated system that can extract real-time product information from Flipkart, store it cleanly, and make it ready for analysis without manual effort.

Key Features

  • Homepage Rendering
  • Dynamic URL Routing
  • HTML Template Rendering
  • Form Handling
  • Easy to Extend

Working of the Project

  • User requests a page
  • FLASK is used for backend efficiency and scalability
  • Hosted on AWS EC2 for scalable performance
  • DyanamoDB ensures real time tracking and personalized user experiences.
  • t is a cloud based platform that ensures seamless ordering, browsing and subscription management

Tools & Technologies Used

  • HTML,CSS
  • Flask Framework
  • JavaScript
  • VS Code

Outcomes

  • DyanamoDB ensures real time tracking and personalized user experiences.
  • Cloud-based scalability and performance.
  • Easy deployment and maintenance.
  • Real-time and accurate product information collected.
  • It is a cloud based platform that ensures seamless ordering, browsing and subscription management.

Advantages
